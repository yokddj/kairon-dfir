# Symbol Fetcher Third-Party Notices

This optional, operator-built image installs Volatility 3 version 2.28.0 from
the official PyPI package using the SHA-256-pinned lock file shared with the
memory worker. Volatility's PDB reader is used only to convert a locally
downloaded, identity-validated PDB into an ISF.

Microsoft PDBs, generated ISFs or populated caches is out of scope and requires
a separate license and third-party-terms review.
