# Questions

Use these questions as a lab guide or review checklist. They are intentionally investigative rather than heavily gamified.

1. What is the hostname of the system?
2. Which user executed the activity?
3. Where was the main lab directory created?
4. Which PowerShell indicators suggest suspicious behavior?
5. Was an encoded PowerShell command executed?
6. Which files were created by the encoded command?
7. Which reconnaissance commands were executed?
8. Was a scheduled task created?
9. What is the scheduled task name?
10. What payload does the scheduled task execute?
11. Was registry-based persistence configured?
12. Which Run Key value was created?
13. What payload does the Run Key execute?
14. Did the Run Key payload execute after logon?
15. Which artifacts support `powershell.exe` execution?
16. Which artifacts support `cmd.exe` execution?
17. Which artifacts support `notepad.exe` execution?
18. What is the most likely sequence of events?
19. Which Kairon DFIR views were most useful to answer the case?
20. Which finding would you prioritize first as an analyst, and why?
